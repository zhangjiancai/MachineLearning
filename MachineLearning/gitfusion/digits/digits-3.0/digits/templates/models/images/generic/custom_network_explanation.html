Specifying a custom Caffe network

You should enter a train_val network here. Some cleanup is done for you automatically:

  • Data layers for the LMDBs in your dataset are created for you automatically
  • If you provide a Data layer here and leave the data_param.source blank, it will be filled in for you with the appropriate path

The deploy network will be created for you automatically. Once caffe implements their "all-in-one" nets feature, you will be able to specify which layers to include in the deploy file. Until then, these things are done automatically:

  • All loss layers and accuracy layers are removed from the deploy file
  • Layers with names starting with train_ are not included in the deploy file
  • Layers with names starting with deploy_ are only included in the deploy file
